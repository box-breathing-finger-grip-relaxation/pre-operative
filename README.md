# Relaksasi Pre-Operatif

Website interaktif untuk membantu pasien preoperatif memahami teknik **Box Breathing** dan **Finger Grip Relaxation** sebagai intervensi nonfarmakologis untuk menurunkan kecemasan dan tekanan darah.

## Fitur
- Animasi Box Breathing (4-4-4-4 detik)
- Simulasi Finger Grip Relaxation (48 detik per jari)
- Timer otomatis 6 menit sesuai SOP penelitian
- Form input tekanan darah dan kuesioner APAIS
- Tampilan sederhana dan mudah digunakan

## Cara Menjalankan
1. Download atau clone repository ini.
2. Buka file `relaksasi-preop.html` di browser.
3. Klik tombol **Mulai Latihan** untuk memulai simulasi.
